# 1. Echive 的 MVP 任务拆解清单

## 1.1 MVP 目标
打通最小闭环：

**想法输入 → AI 澄清 → 任务拆解 → 素材沉淀 → 草稿生成**

- 长期记忆系统

---

## 1.5 验收标准
当你能完成下面这条链路时，MVP 就成立：

1. 输入一个想法
2. 系统生成标题、摘要、澄清问题
3. 系统输出下一步行动
4. 系统生成任务建议
5. 你补充一些素材
6. 系统生成一篇博客提纲或微博草稿
