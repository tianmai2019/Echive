# 2. Echive 的页面原型结构说明

## 2.1 整体导航
左侧主导航：

- Home
- Inbox
- Projects
- Tasks
- Vault
- Studio

顶部区域：

- 全局搜索
- 快速记录按钮
- 用户入口

---

## 2.2 Home 页面

### 目标
作为个人工作台总览。

### 区块
1. **Quick Capture**
   - 一个快速输入框
   - 支持直接记录一句话想法

2. **Today**
   - 今日任务列表
   - 显示优先级和状态

3. **Recent Ideas**
   - 最近输入的想法
   - 可点击进入详情

4. **Pending Clarification**
   - 待澄清的想法
   - 一键进入澄清流程

5. **Recent Drafts**
   - 最近生成的草稿

---

## 2.3 Inbox 页面

### 目标
管理所有新输入内容。

### 页面结构
- 顶部：输入框 + 提交按钮
- 中部：筛选 Tabs
  - All
  - Captured
  - Clarified
  - Planned
- 底部：Idea 卡片列表

### Idea 卡片信息
- 标题
- 摘要
- 状态
- 标签
- 项目归属
- 创建时间
- 下一步操作按钮：
  - Clarify
  - Create Tasks
  - Add Materials
  - Generate Draft

---

## 2.4 Idea 详情页

### 区块
1. 原始输入
2. AI 生成摘要
3. 澄清问题与回答
4. 下一步建议
5. 关联项目
6. 关联任务
7. 关联素材
8. 关联草稿

### 操作按钮
- 开始澄清
- 生成任务
- 添加素材
- 生成草稿

---

## 2.5 Projects 页面

### 目标
管理长期主题或项目。

### 页面结构
- 顶部：新建项目按钮
- 左侧：项目列表
- 右侧：项目详情

### 项目详情
- 标题
- 描述
- 目标
- 状态
- Tabs:
  - Ideas
  - Tasks
  - Materials
  - Drafts

---

## 2.6 Tasks 页面

### 页面结构
- 顶部：视图切换
  - Today
  - This Week
  - All
  - Board
- 列表区：任务列表或看板

### 任务卡片
- 标题
- 来源 Idea
- 所属 Project
- 优先级
- 截止时间
- 状态切换

---

## 2.7 Vault 页面

### 页面结构
- 顶部：新增素材按钮、搜索栏
- 左侧：标签和类型筛选
- 右侧：素材列表

### 素材卡片
- 标题
- 类型
- 摘要
- 来源
- 标签
- 状态
- 关联项目/想法

---

## 2.8 Studio 页面

### 页面结构
- 左侧：Draft 列表
- 右侧：编辑/生成区域

### 生成面板
- 选择来源 Idea
- 选择关联 Materials
- 选择输出格式
  - Blog
  - Weibo
  - Vlog
  - Script
- 生成按钮

### 草稿编辑区
- 标题
- 提纲
- 正文
- 状态
- 保存按钮

---
